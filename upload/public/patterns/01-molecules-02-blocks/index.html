This modal, like the others, can contain various content. The Title is now optional, as is the closing 'X' (but should be present in almost all cases.) The action buttons on the bottom default to a right alignment; however, if a left side button or link is needed (often the case), simply add class 'left' to that link, not the entire container.

Also, note that the cancel link ('button') is a new class "btn__cancel-action". This is important to align the link to the corresponding action 'btn' class button, but without the borders and a new text behavior 'capitalize' vs button 'uppercase'.
